**ChatService.swift** — @Observable @MainActor klass med:

1. **Properties:**
   - `var conversations: [Conversation] = []`
   - `var messages: [ChatMessage] = []`
   - `private var conversationsListener: ListenerRegistration?`
   - `private var messagesListener: ListenerRegistration?`
   - `private let db = Firestore.firestore()`

2. **Konversationslyssnare:**
   - `startListeningToConversations(uid:)` — `db.collection("conversations").whereField("participants", arrayContains: uid).order(by: "lastMessageAt", descending: true).addSnapshotListener` som uppdaterar `conversations`.

3. **Meddelandelyssnare:**
   - `startListeningToMessages(conversationId:)` — `db.collection("conversations").document(conversationId).collection("messages").order(by: "sentAt", ascending: true).limit(toLast: 50).addSnapshotListener` som uppdaterar `messages`.
   - Ta bort forsta lyssnaren med `messagesListener?.remove()` innan ny skapas.

4. **Stop:**
   - `stopListeningToMessages()` — tar bort bara message-listener.
   - `stopAll()` — tar bort bade conversations- och message-listener.

5. **Skicka meddelande:**
   - `sendMessage(_ text: String, conversationId: String, senderId: String) async throws` — addDocument pa messages-subkollektion, sedan updateData pa conversation med lastMessage, lastMessageAt (FieldValue.serverTimestamp()), lastMessageSenderId.

6. **Skicka vaderreaktion:**
   - `sendWeatherSticker(data: WeatherStickerData, conversationId: String, senderId: String) async throws` — skapar ChatMessage med type .weatherSticker och weatherData. Uppdaterar lastMessage till "Vader-sticker" (preview-text).

7. **Hanta/skapa 1-till-1 konversation:**
   - `getOrCreateDirectConversation(currentUid: String, friendUid: String) async throws -> String` — deterministiskt ID: `[currentUid, friendUid].sorted().joined(separator: "_")`. Hamtar dokumentet, skapar om det inte finns.

8. **Skapa gruppkonversation:**
   - `createGroupConversation(creatorUid: String, participantUids: [String], groupName: String?) async throws -> String` — auto-genererat Firestore-ID, isGroup: true. Max 20 deltagare (kasra fel om fler).

9. **Blockering:**
   - `blockUser(uid: String, blockedUid: String) async throws` — setData pa users/{uid}/blockedUsers/{blockedUid} med blockedAt: FieldValue.serverTimestamp().
   - `unblockUser(uid: String, blockedUid: String) async throws` — delete pa samma dokument.
   - `isBlocked(uid: String, blockedUid: String) async -> Bool` — getDocument, returnera doc.exists.
   - `fetchBlockedUserIds(uid: String) async throws -> Set<String>` — hamta alla dokument i blockedUsers-subkollektion, returnera Set av document IDs.

10. **Rapport:**
    - `reportMessage(report: Report) async throws` — addDocument i reports-kollektion.

Anvand `import FirebaseFirestore` och `import Observation`. Folj samma @Observable @MainActor-monster som FriendService.

VIKTIGT: `nonisolated(unsafe) private var conversationsListener` och `nonisolated(unsafe) private var messagesListener` — kravs for deinit-kompatibilitet, precis som i AuthManager.
