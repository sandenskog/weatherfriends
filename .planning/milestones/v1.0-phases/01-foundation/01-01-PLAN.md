    Skapa AuthState.swift:
    ```swift
    enum AuthState {
        case unauthenticated
        case authenticating
        case needsOnboarding
        case authenticated
    }
    ```

    Skapa AuthManager.swift som @Observable @MainActor class:
    - Egenskap: `currentUser: AppUser?`
    - Egenskap: `authState: AuthState = .unauthenticated`
    - Private: `listenerHandle: AuthStateDidChangeListenerHandle?`
    - Init: `addStateDidChangeListener` som sätter authState och anropar fetchOrCreateUserProfile
    - Deinit: `removeStateDidChangeListener`
    - `fetchOrCreateUserProfile(uid:)`: Hämta AppUser från Firestore via UserService. Om profilen saknas (`displayName` är tom eller nil) → sätt `authState = .needsOnboarding`. Annars → sätt `authState = .authenticated`.
    - `signOut()`: Anropa `Auth.auth().signOut()` och nollställ state
    - Lämna login-metoder (signInWithApple, signInWithGoogle, signInWithFacebook) som tomma stubs med TODO-kommentarer — dessa implementeras i Plan 02.

    Skapa AppRouter.swift:
    ```swift
    struct AppRouter: View {
        @Environment(AuthManager.self) private var authManager

        var body: some View {
            switch authManager.authState {
            case .unauthenticated:
                LoginView()      // Placeholder — implementeras i Plan 02
            case .authenticating:
                ProgressView("Loggar in...")
            case .needsOnboarding:
                Text("Onboarding")  // Placeholder — implementeras i Plan 03
            case .authenticated:
                Text("Huvudvy")     // Placeholder — implementeras i fas 2
            }
        }
    }
    ```
    Skapa placeholder LoginView som en enkel SwiftUI-vy med texten "Login" (ersätts i Plan 02).

    Skapa AppUser.swift (Firestore Codable model):
    ```swift
    struct AppUser: Codable, Identifiable {
        @DocumentID var id: String?
        var displayName: String
        var photoURL: String?
        var city: String
        var cityLatitude: Double?
        var cityLongitude: Double?
        var authProvider: String  // "apple" | "google" | "facebook"
        @ServerTimestamp var createdAt: Timestamp?
        @ServerTimestamp var updatedAt: Timestamp?
    }
    ```
    Spara cityLatitude/cityLongitude redan nu (behövs för WeatherKit i fas 2). Inkludera computed property `profileImageURL: URL?`.

    Skapa UserService.swift som @Observable @MainActor class:
    - `createUserProfile(_ user: AppUser)` — setData med merge: true
    - `fetchUser(uid: String) -> AppUser?` — getDocument + data(as:)
    - `updateUser(uid: String, data: [String: Any])` — updateData
    - `uploadProfileImage(uid: String, imageData: Data) -> String` — Firebase Storage upload, returnerar download URL

    Skapa firestore.rules med produktionsregler:
    ```
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /users/{userId} {
          allow read: if request.auth != null;
          allow write: if request.auth != null && request.auth.uid == userId;
        }
      }
    }
    ```
