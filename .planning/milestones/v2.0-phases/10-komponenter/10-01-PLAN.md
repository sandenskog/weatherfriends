  <action>
Skapa ny fil `HotAndColdFriends/DesignSystem/AvatarView.swift`.

AvatarView är en fristående SwiftUI-komponent som:
1. Tar `displayName: String`, `temperatureCelsius: Double?` och `size: CGFloat = 52`
2. Beräknar `TemperatureZone` från celsius (fallback arctic om nil)
3. Renderar en cirkel fylld med `zone.gradient` (LinearGradient, inte solid färg)
4. Visar vita initialer (max 2 bokstäver) ovanpå gradienten — `Text(initials).font(.system(size: size * 0.38, weight: .bold)).foregroundStyle(.white)`
5. Stöder valfri `photoURL: String?` — om URL finns och AsyncImage lyckas, visa foto istället för initialer (cikel-clippat). Om foto misslyckas, visa gradient+initialer.

Struktur:
```swift
struct AvatarView: View {
    let displayName: String
    let temperatureCelsius: Double?
    var size: CGFloat = 52
    var photoURL: String? = nil

    private var zone: TemperatureZone { ... }
    private var initials: String { ... }

    var body: some View { ... }
}
```

Initials-logik: `displayName.split(separator: " ").prefix(2).compactMap { $0.first.map(String.init) }.joined().uppercased()`

Inkludera `#Preview` som visar alla 5 temperaturzoner i en HStack med olika storlekar (40, 52, 64).

Registrera filen i Xcode-projektet via `project.pbxproj` — lägg till under `HotAndColdFriends/DesignSystem`-gruppen. Hitta befintliga DesignSystem-filer i pbxproj och duplicera deras mönster.

  <files>HotAndColdFriends/Features/FriendList/FriendRowView.swift</files>
  <action>
Omforma `FriendRowView.swift` — ersätt den nuvarande list-rad-stilen med ett kort.

Ändringar:
1. Ersätt `initialsCircle`-blocket och `profileImage`-blocket med `AvatarView(displayName: friendWeather.friend.displayName, temperatureCelsius: friendWeather.temperatureCelsius, size: 52, photoURL: friendWeather.friend.photoURL)` — ta bort WeatherAnimationView-lagret (används ej längre i raden, kan finnas kvar i animation-fas 11).

2. Kort-layout: Hela HStack-innehållet ska vara inslaget i ett `.background(Color.bubbleSurface)` → `.clipShape(RoundedRectangle(cornerRadius: CornerRadius.md))` → `.shadowMd()` — kortet ska se ut som ett flytande kort, inte en list-rad.

3. Intern padding: `.padding(.horizontal, Spacing.md)` + `.padding(.vertical, Spacing.sm + Spacing.xs)` (= 12pt vertikalt).

4. Slide-hover: Lägg till `@State private var isPressed = false`. Använd `.onLongPressGesture(minimumDuration: 0, pressing: { pressing in isPressed = pressing }, perform: {})` för att trigga state. Applicera `.scaleEffect(isPressed ? 1.02 : 1.0)` + `(isPressed ? shadowLg() : shadowMd())` med `.animation(.spring(response: 0.3, dampingFraction: 0.7), value: isPressed)`.

5. Typsnitt: Uppdatera `displayName`-texten till `.font(.bubbleH3).foregroundStyle(.bubbleTextPrimary)`, city-texten till `.font(.bubbleCaption).foregroundStyle(.bubbleTextSecondary)`, temperaturvärdet till `.font(.bubbleTemperature).foregroundStyle(zone.color)`.

6. `temperatureCelsius`-konvertering: definiera `private var zone: TemperatureZone { TemperatureZone(celsius: friendWeather.temperatureCelsius ?? -99) }` — använd `.arctic` som fallback.

7. Behåll WeatherIconMapper-ikonen till höger — använd `WeatherIconMapper.icon(for: friendWeather.symbolName, size: 28).foregroundStyle(zone.color)`.

Notera: Ta bort det gamla `initialsCircle`-computed property och hela `profileImage`-ViewBuilder — de ersätts av AvatarView.
