**InstagramStoriesService.swift:**

Skapa `@MainActor class InstagramStoriesService`:

1. `static var canShareToStories: Bool` -- kollar `UIApplication.shared.canOpenURL(URL(string: "instagram-stories://share")!)`. Returnerar false om URL inte kan oeppnas (Instagram ej installerat eller plist saknar schemat).

2. `static func shareToStories(image: UIImage)` -- delar bild som bakgrundsbild:
   - Konvertera image till PNG data
   - Skapa pasteboard items: `["com.instagram.sharedSticker.backgroundImage": imageData]`
   - Saett options med `expirationDate: Date().addingTimeInterval(300)` (5 min)
   - Anropa `UIPasteboard.general.setItems([pasteboardItems], options: options)`
   - Oeppna `instagram-stories://share?source_application=\(Bundle.main.bundleIdentifier ?? "")`

**Info.plist:**

**WeatherCardPreviewSheet.swift:**

Skapa `struct WeatherCardPreviewSheet: View` med:
- `let friendWeather: FriendWeather`
- `@Environment(InviteService.self) private var inviteService`
- `@Environment(\.dismiss) private var dismiss`
- `@State private var renderedImage: UIImage?`
- `@State private var inviteToken: String?`
- `@State private var shareText: String = ""`

Layout (`.sheet` presentation):
- VStack med:
  - Drag indicator oeverst (capsule)
  - WeatherCardView(friendWeather:) som preview, skalad ner med `.scaleEffect(0.7)` eller liknande foer att passa i sheeten. Wrappat i `RoundedRectangle` med shadow.
  - Spacer
  - HStack med tvaa knappar:
    1. **Share-knapp** (primaer): Anvaend `ShareLink` med `item: Image(uiImage: renderedImage!)`, `message: Text(shareText)`, `preview: SharePreview(friendWeather.friend.displayName, image: Image(uiImage: renderedImage!))`. Styled som stor BubblePopButton med "Share" text och SF Symbol `square.and.arrow.up`. Disabled om renderedImage aer nil.
    2. **Instagram Stories-knapp**: Vanlig `Button` som anropar `InstagramStoriesService.shareToStories(image:)`. Visa bara om `InstagramStoriesService.canShareToStories` aer true. Styled med Instagram-gradient bakgrund och kamera-ikon. Label: "Instagram".
  - Padding bottom

`.onAppear`:
- Rendera kortet: `renderedImage = WeatherCardRenderer.renderCard(friendWeather: friendWeather)`
- Haemta invite-token: `inviteToken = try? await inviteService.getOrCreateInviteToken()`
- Generera shareText dynamiskt: "It's [temp] and [condition] in [city] [inviteURL]"

ShareText-format: `"It's \(friendWeather.temperatureFormatted) and \(friendWeather.conditionDescription.lowercased()) in \(friendWeather.friend.city) \(inviteURL)"`
