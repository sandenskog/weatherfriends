**UserService.swift — Ny metod:**

Lägg till `lookupAuthUid(byDisplayName:)` efter befintliga metoder i UserService. Metoden ska:

1. Ta `displayName: String` som parameter
2. Returnera `String?` (Auth UID om match, nil annars)
3. Använda `try?` (inte `throws`) — nätverksfel och no-match returnerar nil
4. Göra en Firestore-query: `db.collection("users").whereField("displayName", isEqualTo: displayName).limit(to: 1).getDocuments()`
5. Returnera `snapshot?.documents.first?.documentID` (document ID = Firebase Auth UID)

```swift
func lookupAuthUid(byDisplayName displayName: String) async -> String? {
    let snapshot = try? await db
        .collection("users")
        .whereField("displayName", isEqualTo: displayName)
        .limit(to: 1)
        .getDocuments()
    return snapshot?.documents.first?.documentID
}
```

Notera: Returnerar nil vid nätverksfel, timeout, eller ingen match. Det är avsett — vännen skapas utan authUid.

    }
}
